## 改版定位

現況問題：第一屏同時出現「多租戶 SaaS、可插拔模組、LIFF、行銷自動化引擎」等詞，企業主還沒理解價值就先感到系統複雜。

參考同業：漸強實驗室、Omnichat 先談「會員經營、回購、轉換、客服效率」再談技術；awoo 把技術能力與顧問經驗放一起建立專業可信度。

**新定位：不是賣一套功能很多的系統，而是協助企業把 LINE 好友變成能持續經營的會員。**
目標形象：「懂 LINE 行銷，也有能力完成系統落地的顧問型科技公司」。

### 問題區塊

**LINE 好友不少，卻沒有真正變成會員？**

很多企業已經經營 LINE 官方帳號，卻仍然遇到這些問題：

- 好友加入之後，不知道他是誰、對什麼有興趣。
- 每次推播都發給所有人，訊息費增加，效果卻不一定提升。
- 會員、預約、點數、票券與消費資料分散在不同系統。
- 活動結束後沒有後續追蹤，辛苦取得的名單慢慢失去聯繫。
- 分店與員工各自管理資料，總部很難掌握實際營運狀況。

真正需要解決的，不只是「怎麼發訊息」，而是如何把每一次互動留下來，成為下一次成交的基礎。

### 解決方案區塊

**把會員經營與日常營運，接進 LINE 裡**

顧客不需要另外下載 App，透過原本熟悉的 LINE，就能完成會員登入、預約、領券、集點、查詢與消費互動。企業則能在後台掌握會員資料與行為，安排更適合的行銷流程。

1. **認識你的會員** — 整合會員資料、標籤、等級、來源與互動紀錄，不再只看到一串沒有分類的好友名單。
2. **在適合的時間主動聯繫** — 依照加入時間、會員標籤、消費狀況、生日或活動日期，自動發送提醒、票券與回購訊息。
3. **讓互動直接走向消費** — 將預約、點數、票券、儲值與商品服務整合進 LINE，縮短顧客從看到訊息到採取行動的距離。
4. **從單店管理到多店營運** — 總部、店長、店員與收銀人員依照權限使用系統，各分店資料清楚分流，會員權益也能依需求跨店使用。

### 宇果優勢區塊

**我們不只提供系統，也協助你把流程規劃好**

很多系統功能很多，真正導入後卻不知道該怎麼用。宇果從企業實際的顧客流程出發，先釐清「客戶從哪裡來、加入後看什麼、如何互動、怎麼成交、何時再次聯繫」，再決定需要導入哪些功能。

1. **顧問式導入** — 不是交付一個空白後台，而是根據你的產業、客群與營運方式，規劃實際可執行的會員流程。
2. **模組化建置** — 只啟用目前需要的功能，降低導入成本與教育負擔。未來增加分店或新服務時，再彈性擴充。
3. **產業流程整合** — 餐飲點餐、美業預約、課程報名、會員票券、POS 收銀與活動系統，都能依照產業情境組合。
4. **客製化串接** — 現有系統無法滿足的欄位、流程或第三方服務，可依實際需求評估 API 串接與客製開發。

### 行銷自動化區塊

**設定一次，該做的會員跟進自動完成**

行銷自動化不是一直傳訊息，而是在適合的時間，對適合的會員說適合的話。例如：

- 新會員加入後，自動發送歡迎訊息與入會好禮。
- 完成問卷後，依照答案自動分類會員標籤。
- 預約日前一天，自動發送提醒，降低臨時取消與未到店。
- 消費一段時間後，自動發送回購券或關懷訊息。
- 會員生日、升等或點數即將到期時，自動通知。
- 不同會員等級，顯示不同的圖文選單與專屬入口。

讓團隊少做重複工作，也避免重要的顧客跟進被遺漏。

按鈕：了解行銷自動化

### 主要功能區塊

**從會員經營到門市營運，一套系統彈性組合**

1. **會員 CRM** — 管理會員基本資料、標籤、群組、等級、互動紀錄與消費狀況。
2. **點數與票券** — 設定集點、兌換、優惠券、禮品券、轉贈與會員專屬活動。
3. **推薦與分享** — 透過推薦碼與分享連結記錄推薦來源，規劃會員邀請與口碑成長活動。
4. **預約與報名** — 整合服務預約、設計師班表、課程報名、活動簽到與提醒通知。
5. **POS 與門市管理** — 依照餐飲、美業與零售需求整合結帳、會員、儲值、庫存與服務紀錄。
6. **多分店管理** — 設定總部、店長、店員與收銀人員權限，讓各店資料分流、總部集中掌握。
7. **問卷與會員標籤** — 顧客填寫資料後，自動依答案貼上標籤，作為後續分眾溝通依據。
8. **抽獎與互動活動** — 支援轉盤、刮刮樂、拉霸、活動報名與票券兌換，讓活動名單回到會員系統中。

### POS 區塊

**預約、會員與結帳，不必各用一套系統**

現場人員在平板完成預約確認、服務選擇、會員折扣、點數與儲值金扣款。每一筆交易都能回到會員資料中，成為後續分眾與回購行銷的依據。

適用情境：

- 美業預約、指定服務人員與加價項目。
- 餐飲點餐、廚房出單與會員優惠。
- 零售結帳、會員折扣與點數累積。
- 多分店會員共用與營運資料管理。

按鈕：了解 POS 與產業模組

- **入門方案** — 適合剛開始建立會員制度的單店品牌。包含會員資料、標籤、等級、點數與基本推播功能。按鈕：了解入門方案

## 品牌用字原則

**多使用**：會員經營、顧客資料、自動跟進、提升回購、減少人工操作、多分店管理、營運流程、依需求導入、顧問協助規劃

**減少使用**：賦能、企業商業增長引擎、全方位解決方案、顛覆傳統、智慧驅動、無縫整合、一站式賦能、AI 革命、打造全新生態系

**AI**：只在確實有 AI 功能的段落出現，不要每一段都把 AI 當成形容詞。

**上線前備註**：案例區最好補上可核實的會員成長、回購、工時或營收數據，說服力比再增加功能介紹更高。
